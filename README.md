# KSU_HM

Python 3.8 설치 후
1. python3 -m pip install --user -U virtualenv
2. 원하는 폴더로 간 후 virtualenv env
3. git clone 프로젝트 주소
4. env/Scripts/activate.bat 실행
5. cd KSU_HM 
6. poetry install (poetry.lock 파일 있는 곳에서)
7. ksu_hm 폴더 안에서 
 python Main.py or </br>
 py Main.py or </br>
 python -m Main or </br>
 py -m Main </br>
